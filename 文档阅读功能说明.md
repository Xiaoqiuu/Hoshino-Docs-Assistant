# 文档阅读功能说明

## 功能概述

现在可以在应用内直接阅读文档，同时保持聊天窗口可见，随时进行 AI 对话。

## 使用方法

### 打开文档阅读器

1. 进入文档库（点击主界面的文档库按钮）
2. **双击**任意已就绪的文档卡片
3. 文档阅读器会在左侧打开，显示文档内容

### 界面布局

```
┌─────────────────────────────────────────────────────┐
│  文档库工具栏                                        │
├──────────────────────────┬──────────────────────────┤
│                          │                          │
│  文档阅读器              │   聊天窗口（右侧）        │
│  - 显示文档内容          │   - 保持可见             │
│  - 支持分页浏览          │   - 随时提问             │
│  - 可关闭               │   - 查看回答             │
│                          │                          │
└──────────────────────────┴──────────────────────────┘
│  悬浮输入框（底部）                                  │
└─────────────────────────────────────────────────────┘
```

### 功能特点

1. **应用内阅读**
   - 无需外部 PDF 阅读器
   - 文档内容直接显示在应用中
   - 按页码组织，清晰易读

2. **同时对话**
   - 阅读文档时，聊天窗口保持在右侧
   - 可以边看边问，无需切换界面
   - 输入框始终可用

3. **便捷操作**
   - 双击打开文档
   - 点击关闭按钮返回文档列表
   - 支持分页浏览（如果文档有多页）

## 操作提示

- **单击**文档卡片：选择/取消选择文档（用于提问）
- **双击**文档卡片：打开文档阅读器
- 阅读器右上角的 **✕** 按钮：关闭阅读器，返回文档列表
- 聊天窗口可以最小化或关闭，不影响阅读器使用

## 技术实现

- 文档内容从向量数据库中提取
- 按页码自动组织和排序
- 支持大文档的流畅显示
- 粉色主题统一设计
